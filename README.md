# Target Sprint

A small Flask-backed mini game built with HTML, Tailwind CSS, and basic JavaScript.
hshdgfhksdhf

## Run locally

1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Run `python app.py`.
4. Open `http://127.0.0.1:5000`.

## Game flow

- Click Start Game.
- Tap the target as many times as possible before the timer ends.
- Scores are stored by the Flask backend and shown in the leaderboard.
